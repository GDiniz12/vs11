# 16-0 ⚽ Football Simulation Game

A premium football management simulation game built with Next.js 14, React 18, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to play.

## Features

- 🏟️ Choose from 3 formations (4-3-3, 4-4-2, 3-4-3)
- ⚽ Draft players from real historic teams
- 🏆 Champions League-style league phase (36 teams)
- 🥅 Knockout rounds (R16, QF, SF, Final)
- 📊 Realistic match simulation (Poisson-based)
- 🎨 Premium, modern UI design

## Tech Stack

- **Next.js 14** (App Router)
- **React 18**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
